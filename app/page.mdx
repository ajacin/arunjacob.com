# Arun Jacob

I'm a senior frontend engineer with over 10 years of experience, based in Woodstock, Ontario. I build fast, accessible product interfaces, mostly with React and TypeScript, across healthcare and fintech.

Outside of employment, I run [Lux Compose](https://luxcompose.com), a small indie studio. My current project is [GlassNudge](https://glassnudge.app), a macOS menu bar reminder app built with Tauri v2 and Rust. I also write [FedNews](https://fednews.fyi), a daily newsletter for senior frontend developers.

Originally from India, now navigating Ontario winters and the occasional Rust borrow checker error.
